| Classe exception                   | Lançado quando                                                                                          | Mensagem                                                                                | Parent                  |
|------------------------------------|---------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|-------------------------|
| `BusinessException`                | Todas as exceções personalizadas                                                                        | *Houve um erro inesperado*                                                              | `RuntimeException`      |
| `ModelException`                   | Todas as exceções de model                                                                              | *Houve um erro inesperado na tentativa de realizar a ação*                              | `BusinessException`     |
| `PatchException`                   | Todas as exceções de patch                                                                              | *Houve um erro inesperado na tentativa da atualização*                                  | `BusinessException`     |
| `TransactionException`             | Todas as exceções de transação                                                                          | *Houve um erro inesperado na tentativa de transação*                                    | `BusinessException`     |
| `ValidationException`              | Todas as exceções de validação                                                                          | *Houve um erro no cadastro de dados*                                                    | `BusinessException`     |
| `ActiveModelDeleteException`       | Na tentativa de excluir model ativo                                                                     | *Não é possível excluir **classe** se for ativo*                                        | `ModelException`        |
| `InactiveResourceException`        | Na tentativa de fazer transações em model inativo                                                       | ***Classe** não pode sofrer alterações já que está inativado*                           | `ModelException`        |
| `MinimumInactivityPeriodException` | Na tentativa de excluir models inativos que ainda não passaram do tempo máximo de permanência (6 meses) | *Não é possível excluir dados com menos de seis meses de inatividade (**meses** meses)* | `ModelException`        |
| `ModelNotFoundException`           | Na tentativa de busca de model inexistente                                                              | *Não foi possível encontrar **classe***                                                 | `ModelException`        |
| `ImmutableFieldException`          | Na tentativa de patch de campo imutável                                                                 | *Campo **campo** não pode ser alterado*                                                 | `PatchException`        |
| `InexistentFieldPatchException`    | Na tentativa de patch de campo inexistente                                                              | *Campo **campo** não existe em **classe***                                              | `PatchException`        |
| `UniqueViolationException`         | Na tentativa de cadastro de valor que já está no banco                                                  | *Campo com o valor **valor** já está cadastrado*                                        | `PatchException`        |
| `InsufficientBalanceException`     | Na tentativa de transferência, cujo saldo ficará menor que R$-200,00                                    | *O limite de R$-200,00 foi atingido e não foi possível terminar a transação*            | `TransactionException`  |
| `NegativeOrZeroTransferException`  | Na tentativa de transferência de valor menor ou igual a zero                                            | *A transferência não pode ser de um valor menor que zero*                               | `TransactionException`  |
| `SameAccountTransferException`     | Na tentativa de transferência para a mesma conta                                                        | *Não é possível fazer uma transação para sua própria conta*                             | `TransactionException`  |
| `ZeroAmountException`              | Na tentativa de atualização de saldo de zero reais                                                      | *Não é possível fazer uma transação de R$0,00*                                          | `TransactionException`  |
| `InvalidAgeException`              | Na tentativa de cadastro de cliente com idade errada                                                    | *Não é possível cadastrar pessoas com a idade registrada*                               | `ValidationException`   |
| `InvalidInputException`            | Na tentativa de input errado                                                                            | *Campo **campo** digitado não é válido*                                                 | `ValidationException`   |
| `InvalidAccountTypeException`      | Na tentativa de patch de tipo de conta inexistente                                                      | *Campo **tipo** digitado não é válido*                                                  | `InvalidInputException` |
| `InvalidCPFException`              | Na tentativa de input de CPF inexistente                                                                | *Campo **CPF** digitado não é válido*                                                   | `InvalidInputException` |
| `InvalidDDDException`              | Na tentativa de input de DDD inexistente                                                                | *Campo **DDD** digitado não é válido*                                                   | `InvalidInputException` |
| `InvalidEmailException`            | Na tentativa de input de e-mail errado                                                                  | *Campo **e-mail** digitado não é válido*                                                | `InvalidInputException` |
| `InvalidPhoneException`            | Na tentativa de input de telefone inexistente                                                           | *Campo **telefone** digitado não é válido*                                              | `InvalidInputException` |
| `UnderageException`                | Na tentativa de cadastro de menor de idade                                                              | *Não é possível cadastrar pessoas com menos de 18 anos*                                 | `InvalidAgeException`   |
| `UnrealisticAgeException`          | Na tentativa de cadastro de maior de 120 anos                                                           | *Não é possível cadastrar pessoas com mais de 120 anos*                                 | `InvalidAgeException`   |
